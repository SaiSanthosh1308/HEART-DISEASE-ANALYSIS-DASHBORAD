# ❤️ Heart Disease Analysis Dashboard (Excel Project)

## 🔍 Overview

This project presents an interactive **Heart Disease Analysis Dashboard** built using Microsoft Excel. The dataset contains patient medical records used to analyze risk factors and patterns associated with heart disease.

The goal of this project is to explore how different health indicators contribute to the **presence or absence of heart disease**, enabling better understanding and early risk identification.

---

## 📁 Dataset Description

The dataset includes clinical and demographic information of patients with the following features:

* **Age** – Age of the patient
* **Gender** – Male/Female
* **Chest Pain Type** – Type of chest pain (ATA, NAP, ASY, etc.)
* **BP (Blood Pressure)** – Resting blood pressure
* **Cholesterol** – Serum cholesterol level
* **FBS over 120** – Fasting blood sugar indicator (>120 mg/dl)
* **EKG Results** – Electrocardiographic results
* **Max HR** – Maximum heart rate achieved
* **Exercise Angina** – Exercise-induced chest pain
* **ST Depression** – Depression induced by exercise relative to rest
* **Slope of ST Segment** – Slope of peak exercise ST segment
* **Number of Vessels (Fluoroscopy)** – Major vessels colored
* **Thallium Test** – Thallium stress test result
* **Heart Disease** – Target variable (Presence / Absence)

---

## 📊 Dashboard Features

* 📈 Analysis of heart disease occurrence by age and gender
* ❤️ Risk factor comparison (BP, cholesterol, heart rate)
* 🧪 Medical test result analysis (ECG, Thallium, ST depression)
* 🏥 Patient segmentation based on symptoms
* 📊 Distribution of heart disease presence vs absence
